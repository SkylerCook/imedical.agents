# iris-interface 候选资产清单

本清单用于约束 `tmp/his-interface-agent` 来源工程资产的并入方式。

| 分类 | 来源资产 | 处理 |
|---|---|---|
| 保留并改写 | 文档解析数据结构、DOCX/PDF/XLSX 表格抽取、表头映射 | 进入脚本和轻量运行时。 |
| 保留并改写 | 字段归一化、多轮匹配诊断、未匹配字段报告 | 进入字段诊断能力。 |
| 保留并改写 | 代码质量分析中的字段覆盖、TODO、无效表达式、点号循环风险 | 进入离线 review 脚本。 |
| 候选 wiki | HIS 数据流、MOC、接口索引格式、踩坑经验 | 放 `references/wiki/`，不生成 thin-index。 |
| 候选审查 | `rules/domains/*.md`、`rules/common/*.json` | 逐条脱敏和质量修复后再决定是否沉淀。 |
| 丢弃 | 写死本地路径或具体输入文档的脚本 | 不进入插件。 |
| 禁止进入仓库 | 项目连接事实、接口注册事实、具体包路径映射、真实环境测试数据策略 | 仅允许目标项目本地配置。 |
